# Ресурсы игры (assets)

Ресурсы (3D-модели, текстуры, звуки, музыка, видео) **не хранятся в git** —
они слишком большие: всего ~429 МБ, а видео меню `Меню.mp4` (~224 МБ) одно
превышает лимит GitHub в 100 МБ на файл.

## Где взять

Скачайте установщик из [Releases](https://github.com/fedor655/Patisson-Game/releases)
и установите игру — все ресурсы окажутся в папке установки рядом с `Patisson.exe`.
Чтобы запускать из исходника, скопируйте эти файлы в одну папку с `patisson.py`.

Игра при старте делает `os.chdir` в свою папку и грузит ресурсы по относительным
именам, поэтому код и ресурсы должны лежать вместе.
