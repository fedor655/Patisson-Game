# 🎃 Patisson Game

3D-игра-ферма с рыбалкой, написанная на **Python** с движком
[**Ursina**](https://www.ursinaengine.org/) (поверх Panda3D).

Выращивайте патиссон, ловите рыбу, развивайте ферму — и веселитесь!

> **Слова разработчика (Богдан):**
> «Вот и дождались! Релиз Patisson Game! В игре могут быть баги. Игра сделана на
> python. В отличие от первой моей игры (Pacman), эта игра в разы интереснее.
> Веселитесь!»

📅 **Релиз: 25 августа 2025.**

## 🎮 Как играть

Самый простой способ — скачать установщик из раздела
[**Releases**](https://github.com/fedor655/Patisson-Game/releases) и установить игру.

### Управление
| Действие | Клавиша |
|----------|---------|
| Движение | `W` `A` `S` `D` |
| Прыжок | `Space` |
| Взаимодействие | `E` |
| Лейка / Удобрение / Удочка / Саженец (инвентарь) | `1` `2` `3` `4` |
| Переключить интерфейс | `F1` |
| Меню паузы | `Esc` |
| Консоль читов | `F12` |

### Игровой процесс
- 🌱 Сажайте и поливайте патиссон, удобряйте его — он растёт по стадиям.
- 🐟 Ловите рыбу у пруда и продавайте её.
- 🪣 Набирайте воду в колодце (зажмите ЛКМ на рукояти и крутите мышь).
- 🛒 Покупайте улучшения в магазине (золотая лейка, зачарованная удочка и др.).
- 🌗 День сменяется ночью.

## 🕹️ Читы

Откройте консоль клавишей **`F12`** и введите код. Формат: `Команда: значение`.

| Чит | Эффект |
|-----|--------|
| `GiveMoney: <число>` | Даёт указанное количество монет |
| `PatissonUP: <проценты>` | Мгновенно растит патиссон на N% (макс. 100) |
| `SpeedGrowth: <число>` | Увеличивает скорость роста в N раз |
| `ImmortalPlant` | Патиссон не умирает от нехватки воды/еды (бессмертие) |
| `MortalPlant` | Отключает бессмертие |
| `CameraON` / `CameraOFF` | Включает/выключает свободную камеру |
| `SetSensitivity: <число>` | Чувствительность мыши для свободной камеры |

## 🛠️ Запуск из исходного кода

```bash
pip install -r requirements.txt
python patisson.py
```

> ℹ️ Авторский оригинал `patisson.py` добавляется в `main`. Уже сейчас доступна
> **побайтово проверенная восстановленная версия** исходника — в ветке
> [`decompilation`](https://github.com/fedor655/Patisson-Game/tree/decompilation/decompilation).

Игре нужны **ресурсы** (3D-модели `.glb`, текстуры, звуки, музыка, видео меню) —
они не хранятся в репозитории из-за размера (видео меню одно весит ~224 МБ, что
больше лимита GitHub в 100 МБ на файл). Скачайте их из установщика в
[Releases](https://github.com/fedor655/Patisson-Game/releases) и положите рядом с
`patisson.py`. Подробнее — в [`ASSETS.md`](ASSETS.md).
